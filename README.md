# Hackathon Web

Projeto desenvolvido para o hackathon.

## Estrutura inicial

- `app/Controllers`: controladores da aplicacao
- `app/Models`: modelos de dominio
- `app/Views`: paginas e componentes visuais
- `config`: configuracoes da aplicacao
- `core`: infraestrutura e classes base
- `public`: arquivos publicos, estilos, scripts e assets

O codigo da aplicacao sera versionado gradualmente em commits posteriores.
